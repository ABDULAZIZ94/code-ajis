# CodeAjis

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.1.

## install fontaawosome 0.6.1

    ng add @fortawesome/angular-fontawesome@0.6.1

## create social-login component and module
component

    ng g c social-login/social-login

module

    ng g c social-login/social-login


## create tutorials component and module
component

    ng g c tutorials/tutorial

module

    ng g m tutorials/tutorial --flat --routing
    
## Development server

    sudo ng serve --host 0.0.0.0 --port 80
