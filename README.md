# CodeAjis

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.1.

## install fontaawosome 0.6.1

    sudo ng add @fortawesome/angular-fontawesome@0.6.1

## install angular/fire and firebase

    sudo ng add @angular/fire

    sudo npm install --save firebase

## create social-login component and module
component

    ng g c social-login/social-login

module

    ng g c social-login/social-login


## create tutorials component and module
component

    ng g c tutorials/tutorial

module

    ng g m tutorials/tutorial --flat --routing
    
## Development server

    sudo ng serve --host 0.0.0.0 --port 80

## Add nebular theme

    sudo add

## git tag and branch
    
add new tag

    git tag -a <v-x.x.x> -m "tag msg'

    git push origin v-x-x

show tag

    git tag

add branch

    git branch <branch-name>

git switch branch

    git checkout <branch-name>

delete branch 

    git branch -d <branch-name>

show branch
    
    git branch -a

    git branch -r

    git show-branch

## build for production

    ng build --prod
